- Change to `public sealed class Address : Entity<AddressId>`

- Change `private Address()` to `protected Address() : base() { }` (update the string.Empty initializers to remain in the body)

No EF Core migration needed for ADOPT-01 — the Entity<TId> base class only adds an Id property that already exists in the database schema.
