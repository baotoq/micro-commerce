# Phase 16.1 Plan 01: Vogen Adoption for All Strongly-Typed IDs Summary

**One-liner:** Migrated all 14 strongly-typed ID types from hand-rolled StronglyTypedId<T> class hierarchy to Vogen 8.0.4 source-generated partial record structs with Guid.Empty validation, v7 UUID factories, and source-generated EF Core converters.

## What Was Built

Replaced the hand-rolled `StronglyTypedId<T>` base class and reflection-based `StronglyTypedIdConvention` with Vogen's source generator. All 14 ID types are now zero-overhead value types with compile-time safety, built-in validation, and proper JSON/EF Core serialization.

**Before:**
```csharp
public sealed record ProductId(Guid Value) : StronglyTypedId<Guid>(Value) {
    public static ProductId New() => new(Guid.NewGuid());
    public static ProductId From(Guid value) => new(value);
}
```

**After:**
```csharp
[ValueObject<Guid>(conversions: Conversions.EfCoreValueConverter | Conversions.SystemTextJson)]
public partial record struct ProductId {
    public static Validation Validate(Guid value) =>
        value != Guid.Empty ? Validation.Ok : Validation.Invalid("ProductId cannot be empty.");
    public static ProductId New() => From(Guid.CreateVersion7());
}
```

## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 1 - Bug] Fixed null! initializers on struct ID properties in entity classes**
- **Found during:** Task 2 (build phase)
- **Issue:** 7 entity files used `= null!` initializer on ID properties (valid for reference types, invalid for value types). After changing IDs from record classes to record structs, these became compile errors: `Cannot convert null to 'XxxId' because it is a non-nullable value type`
- **Fix:** Removed `= null!` initializers from ID properties in: Product.cs, CartItem.cs, OrderItem.cs, StockReservation.cs, StockAdjustment.cs, WishlistItem.cs, Address.cs. Structs default-initialize to their zero value (Guid.Empty) which is EF Core's expected behavior for shadow-loaded entities.
- **Files modified:** 7 entity/value-object files
- **Commit:** 56993b74
