## Decisions Made
- Kept single `name` field (Full Name) instead of splitting to first/last name -- changing ShippingAddressDto would require backend API changes (Rule 4 scope)
- Added `id="shipping-form"` to shipping form to fix bug where checkout-page referenced `form="shipping-form"` but no form had that id
- Removed internal submit button from shipping section since checkout-page handles navigation buttons externally
- Used `<nav>` element for step indicator instead of `<div>` to properly support `aria-label` attribute per Biome a11y rules
